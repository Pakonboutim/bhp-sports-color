⚙️ การเชื่อมต่อ Google Sheet

📌 วางลิงก์ Google Apps Script Web App ที่นี่ (ดูคู่มือ Code.gs)

คู่มือตั้งค่าครั้งแรก
1. เปิด Google Sheet ใหม่ (หรือเปิดที่มีอยู่)
2. เมนู Extensions → Apps Script
3. วางโค้ดจากไฟล์ Code.gs ทั้งหมด
4. รัน setupSheets() 1 ครั้ง (จะสร้าง Sheet ให้อัตโนมัติ)
5. Deploy → New deployment → Web app
   • Execute as: Me
   • Who has access: Anyone
6. คัดลอก URL มาวางที่ช่องด้านบน
7. กรอกรายชื่อนักเรียนในชีต "นักเรียน"
